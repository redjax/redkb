# RedKB Documentation

RedKB is my personal knowledgebase, where I keep notes about things I learn and want to reference later or share with others. The site is built with [Hugo](https://gohugo.io), and releases to Github Pages. The [`https://redkb.fyi` domain](https://redkb.fyi) points to the Github Pages instance.
