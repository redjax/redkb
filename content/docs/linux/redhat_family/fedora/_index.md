[Fedora](https://fedoraproject.org/) is developed by the Fedora project, and sponsored by RedHat. It follows its own release schedule with a new version every ~6 months. Historically, Fedora was the "testbed" for RedHat Enterprise Linux, where users could test new changes that would eventually land in the enterprise offering.

Check the Fedora documentation for [a history of the relationship between RedHat Enterprise Linux and Fedora](https://docs.fedoraproject.org/en-US/quick-docs/fedora-and-red-hat-enterprise-linux/).
