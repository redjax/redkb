# RedKB - Personal Knowledgebase

Source code for [https://redkb.fyi](https://redkb.fyi)

## What?

This repository is a collection of my notes and code snippets, presented in technical documentation form. The site is built with [Hugo](https://gohugo.io) and the [Hextra theme](https://github.com/imfing/hextra).

Read the [documentation](./docs) for details about how the site is built and deployed.

## Why?

As a lifelong computer enthusiast and professional who values documentation, I have taken notes on problems/fixes, tools, and frameworks I us. I have pages and pages of notes spread across every knowledge management system I've tried, and wanted a place to distill & centralize the documentation I find most useful, or want to share with others.
